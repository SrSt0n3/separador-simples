# Separador Simples

## Como usar

### Instalar
```
npm i fs
```

### config.json
```
{
    "cortes": 2, // numero de arquivos por pasta
    "arquivos": "todos",  // pasta que vai ta os arquivos para separar
    "separados": "separados" // pasta que vao ir os arquivos separados
}
```

### Rodar o projeto
```
node index.js
```
